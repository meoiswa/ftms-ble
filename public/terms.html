Description of the application

FTMS Dashboard is a free, open-source web application that connects to Bluetooth Low Energy fitness machines supporting the Fitness Machine Service (FTMS) profile, displays real-time workout data, and stores session history locally in your browser. An optional Google Drive integration is available for cloud backup and cross-device sync.
