FTMS Dashboard

A browser-based dashboard for Bluetooth fitness machines. Connect, track, and review your workouts — no app install, no account required.

Open Dashboard →

Features

📡

Bluetooth Connect

Pairs directly with any FTMS-compatible fitness machine via the Web Bluetooth API — no drivers, no cables.

📊

Live Dashboard

Real-time display of speed, power, cadence, heart rate, distance and calories with a rolling 30-second chart.

⏺

Session Recording

Records full workout sessions automatically when movement is detected, or start manually with one tap.

📁

Session History

Browse and review past sessions with summary stats and a full chart replay. Stored locally in your browser.

☁️

Google Drive Sync

Optionally back up sessions to your Google Drive and sync across devices. Only files created by this app are accessed.

🔒

Privacy First

No servers. No tracking. No accounts needed. All data stays in your browser unless you opt in to Drive sync.

How We Use Google Account Data

Google Drive Integration — Optional

If you choose to sign in with Google, the app requests one permission scope:

https://www.googleapis.com/auth/drive.file

This scope grants access only to files that this app itself creates. It cannot read, modify, or delete any other files in your Google Drive.

This permission is used exclusively to:

  • Save recorded workout sessions as JSON files in a FTMS Dashboard folder in your Drive
  • Download those files to sync sessions across browsers or devices
  • Delete a session's file when you delete it in the app

Your Google credentials are never stored by this app. The access token is held in memory only and discarded when you sign out or close the tab. The app has no backend server — all Drive requests go directly from your browser to Google's API.
